## How to Contribute

1. Fork the repository
2. Create a feature branch (`git checkout -b feature/amazing-solution`)
3. Write your solution following the project structure
4. Add tests for your solution
5. Ensure all tests pass (`gmake test`)
6. Commit your changes with conventional commits
7. Push to your branch
8. Open a Pull Request
