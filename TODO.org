* Algorithm Patterns Reference
** Graph Algorithms
- BFS :: Shortest path in unweighted graphs
- Dijkstra :: Shortest path with weighted edges
- A* :: Heuristic-guided pathfinding
- Topological Sort :: DAG ordering

** Grid Patterns
- Flood fill :: Connected component detection
- Direction arrays :: 4-way and 8-way movement
- Rotation/reflection :: Grid transformations

** Math Patterns
- GCD/LCM :: Common in cycle detection
- Chinese Remainder Theorem :: Modular arithmetic puzzles
- Memoization :: Dynamic programming optimization
